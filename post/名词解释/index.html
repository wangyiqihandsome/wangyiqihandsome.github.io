名词解释

1、

潮汐：潮汐现象是指海水在天体（主要是月球和太阳）引潮力作用下所产生的周期性运动，习惯上把海面铅直向涨落称为潮汐（海水在水平方向的流动称为潮流）

2、潮差：

相邻的高潮高与低潮高之差

3、平太阳日：

假定一个平太阳在天赤道上（不是黄道）坐等速运行，其速度等于运行在黄道上真太阳的平均速度，这个假象的太阳连续两次上中天的时间间隔叫做一个平太阳日，（把1/24平太阳日取为1平太阳时）

4、风暴潮：是来自海上的一种巨大自然灾害现象，指由于强烈的大气扰动——如强风和气压骤变所招致的海面异常升高现象
5、朔望月：月球从新月（或满月）位置出发再回到新月（或满月）位置的时间间隔
6、正规半日潮：在一个太阴日内（24小时50分）有两次高潮和低潮，两次潮差基本相等
7、不正规半日潮：一个朔望月中大多数时间内是半日潮，少数日子（月赤纬较大）成日潮特征
8、正规日潮：在一个太阴日内（24小时50分）有一次高潮和低潮
9、不正规日潮：一个朔望月中大多数时间内是日潮，少数日子（月赤纬很小）成半日潮特征
10、潮汐不等现象：一天中两个潮差不等，涨潮时和落潮时也不等
11、中尺度涡：水平尺度100-500km、时间尺度20-200d、速度约1-5cm/s的流涡
12、八分潮算法：农历初一、十六，高潮时（低潮时）就等于该地高潮间隙（低潮间隙），然后每天落后0.8小时
高潮时 = 0.8*[农历日期1(16)]+高潮间隙
低潮时 = 0.8*[农历日期1(16)]+低潮间隙
13、海洋水团：源地和形成机制相近，具有相对均匀的物理、化学和生物特征及大体一致的变化趋势，而与周围海水存在明显差异的宏大水体
14、岩石圈：是指软流圈之上的刚性固体物质层，包括地壳和上地幔顶部的刚性岩层，是一个力学概念，具有力学上的统一性和实在性，可以对机械应力作出刚性反应 15、海底峡谷：是陆坡上的一种奇特侵蚀地形，形如深邃的凹槽切蚀于大陆坡上，横剖面通常为不规则的V型，下切深度数百米甚至上千米，谷壁最陡40°以上，与陆上河谷极为相似 16、海洋次级生产量：海洋次级生产量是除生产者外的其它有机体的生产，即消费者和分解者利用初级生产量进行同化作用，表现为海洋动物和其它异养生物生长、繁殖和营养物质的贮存 17、恢复生态学：是研究生态系统退化的原因、退化生态系统恢复与重建的技术和方法及其生态学过程和机理的学科 18、天然气水合物：20世纪科学考察中发现的一种新的矿产资源，它是水分子和碳氢气体在高压和低温条件下混合时产生的一种冰晶状固体化合物，外貌极像冰雪或固体酒精，点火即可燃烧，有“可燃水”、“气冰”、“固体瓦斯”之称 19、海水状态方程：海水状态参数温度、盐度、压力与密度或比容之间相互关系的数学表达式，也称之为p-V-t关系。 一个大气压国际海水状态方程： $$\rho (s,t,0) = \rho \omega + AS + BS^{\frac{2}{3} } + CS^{2}$$ 20、对流层的气温分布特征：温度的铅直分布主要由大气与地、海面热量交换以及大气的对流、湍流运动决定，总的趋势是温度随高度降低，平均递减率约为6.5K/km。温度随高度降到-50~-70°C左右，再向上，温度的降低趋缓慢或向上稍有增加，当温度递减率减小到2K/km（或更小）的最低高度，就规定为对流层的层顶，大气的铅直混合强，气象要素水平分布不均匀。 21、海洋污损生物：也称为海洋附着生物，是指生长在船底和海中一切设施表面的动物、植物和微生物 22、海洋钻孔生物：在海洋中，能穿凿木船、木竹建筑、红树、岩石、珊瑚礁以及贝壳等物体基质的生物 23、初级生产力：浮游植物、底栖植物以及自养细菌等生产者，通过光合作用或化学合成制造有机物和固定能量的能力 24、赤潮：是海洋中某些微小的浮游藻类、原生动物或细菌，在一定的环境条件下爆发性繁殖（增殖）或聚集而引起水体变色的一种有害的生态异常现象 25、生物多样性：是所有生物种类，种内遗传变异和它们的生存环境的总称，包括所有不同种类的动物、植物和微生物，以及它们所拥有的基因，它们与生存环境所组成的生态系统。通常包括遗传多样性、物种多样性和生态系统多样性三个层次
26、卫星海洋遥感：或称为空间海洋学，是利用电磁波与大气和海洋的相互作用原理，从卫星平台观测和研究海洋的分支学科
27、被动传感器：以被动地接收观测对象发射和散射的电磁波（包括可见光、红外和微波等）来探测目标对象的传感器
28、GIS系统：即地理信息系统，是以采集、存储、管理、描述和分析与地球表面及空间地理分布有关的数据的信息系统，它是以地理空间数据库为基础，在计算机硬件、软件环境支持下，对空间相关数据进行采集、管理、操作、分析、模拟和显示，并采用地理模型分析方法，适时提供多种空间和动态的地理信息，为地理研究、综合评价、管理、定量分析和决策服务而建立的一类计算机应用系统
29、SeaWiFS:是装在在美国SEASTAR卫星上的第二代水色遥感传感器，共有8个通道，地面分辨率1.1km，刈幅宽度1502-2801km，观测角沿轨迹方向倾角为20°、0°、-20°。10bit量化
30、辐射量定标：海色传感器输出的计数值卄千，并非真正意义上的物理量，必须利用标准源将计数值换算成辐亮度，这一换算过程就叫做辐射量定标
31、SAR：即合成孔径雷达，是一种主动式微波成象雷达，它被认为是最有效、最有潜力的卫星传感器。具有良好的空间分辨率和全天候全天时工作等优点。利用微波和海面微尺度结构的相互作用而形成海面回向散射系数σ0的图象分布对海洋进行观测
32、地转流：在不考虑海水的湍应力和其它能够影响海水流动的因素的情况下，水平压强梯度力与科氏力的作用，大小相等方向相反取得平衡时的海水定常流动。
33、风海流：由风引起的海流称为风海流或漂流
34、上升流：上升流是指海水从深层向上涌升
35、热盐环流：由温、盐变化引起的环流。相对风生流而言其流动缓慢，是形成大洋中下层温、盐分布特征及海洋层化结构的主要原因
36、海洋中的正压场：在静态的海洋中，当海水密度为常数或者只是深度的函数时，海洋中压力的变化也只是深度的函数，此时海洋中的等压面必然是水平的，即与等势面平行
37、气压：大气压强简称气压，指观测高度到大气上界单位面积上铅直空气柱的重量
38、大气对流层（解释其基本特征）：对流层是大气的最低层，下界是地球表面，上界是对流层顶。对流层的基本特征是：温度随高度降低；大气的铅直混合强；气象要素水平分布不均匀
39、季风：季风是大范围盛行风向随季节有显著变化的风系。主要是由于海陆温度对比的季节性变化和地球上行星风系的季节性南北移动所致
40、热容：海水温度升高1K或1°C时所吸收的热量
41、位温：海洋中某一深度（压力为p）的海水微团，绝热上升到海面（压力为大气压$p_{0}$）时所具有的温度称为该深度的位温
42、（海水的）密度：单位体积海水的质量用符号$ \rho $表示，单位是$kg/m^{3}$
43、海洋锋：指性质明显不同的两种或几种水体之间的狭窄过渡带。狭义而言是水团之间的边界线，广义而言是指任一种海洋环境参数的跃变带，如水温锋、盐度锋、密度锋、声速锋、水色锋、透明度锋，以及海水化学、生物等等要素的海洋锋的称谓
44、盐度：近似地表示海水中所含盐类总量，大洋海水的平均盐度为35
45、海流：指海水大规模相对稳定的流动，是海水重要的普遍运动形式之一
46、重力波：海洋中存在各种不同周期的波动，对短波而言，例如风浪、涌浪只考虑重力对它们的作用，故称为重力波，地转效应可以忽略不计
47、波周期：相邻两波峰（或波谷）通过某固定点所经历的时间
48、海洋内波：在海洋内部发生的波动现象
49、风浪：指当地风产生，且一直处在风的作用之下的海面波动状态
50、空间平台：装载传感器的空间运载工具称为空间平台，包括人造卫星宇宙飞船天空实验室等
